# SDTKCODE Website Setup Checklist

## Document Role

This is the website and product-page follow-up checklist for the public npm package `sdtk-code-kit`.

It is not proof that a dedicated website rollout is already complete.

## Domain & Hosting

- [ ] Subdomain or path allocated on AgentToolkits site
- [ ] SSL certificate configured
- [ ] DNS records set

## Product Page

- [ ] Product listed in AgentToolkits catalog
- [ ] Product metadata matches `agenttoolkits.product.json`
- [ ] Tagline displayed: "Coding process discipline: TDD, debugging, verification, review, and shipping."
- [ ] Installation instructions published
- [ ] Runtime support matrix visible

## Documentation

- [ ] Usage guide published
- [ ] Installation runbook linked
- [ ] API/CLI reference available
- [ ] Changelog accessible

## Analytics & Monitoring

- [ ] Page view tracking enabled
- [ ] Download/install metrics tracked
- [ ] Error monitoring configured

## Status

This checklist is active follow-up documentation for the public `SDTK-CODE` package surface.

It does not change the current truth that:
- the npm package is public
- a dedicated product-page rollout is still follow-up work
