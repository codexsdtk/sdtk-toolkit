- **Public Package**: `sdtk-code-kit`

## First-Public Release Truth

The first go-live claim for this product is narrow and practical:
- the npm package is publicly installable
- the command surface is real
- the package remains downstream of `SDTK-SPEC`

This blueprint does not claim:
- a standalone marketing launch
- a standalone product-site rollout
- independence from upstream `SDTK-SPEC` artifacts

## Go-Live Prerequisites

- [ ] package metadata is public-ready
- [ ] package README and governance docs match real package behavior
- [ ] tarball and installed-package smoke pass
- [ ] maintainer-run publish succeeds
- [ ] post-publish verification evidence is captured
- [ ] product metadata remains aligned with `agenttoolkits.product.json`
