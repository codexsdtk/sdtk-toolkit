# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a Vulnerability

Please report security issues privately.

- Email: security@sdtk.local
- Include: impact, steps to reproduce, affected files/flows, and suggested mitigation.

Expected response timeline:
- Acknowledgement: within 3 business days
- Initial assessment: within 7 business days
- Remediation plan: based on severity and release window

Please do not disclose vulnerabilities publicly until a fix is released.
